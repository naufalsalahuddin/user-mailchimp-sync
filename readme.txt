WordPress Mailchimp Sync Plugin
Overview
The WordPress Mailchimp Sync Plugin is a powerful tool designed to seamlessly integrate your WordPress website with Mailchimp, allowing you to synchronize user data effortlessly. By automating the process of updating user information, this plugin ensures that your Mailchimp audience remains accurate and up-to-date.

Features
Automatic Sync: Syncs user data to Mailchimp in real-time whenever a new user registers or an existing user updates their profile.
Customizable Settings: Easily configure Mailchimp API key and List ID through the WordPress admin panel.
User-friendly Interface: Simple and intuitive settings interface for easy setup and management.

Installation
1. Download the plugin zip file from the Releases section.
2. Log in to your WordPress admin panel.
3. Go to Plugins > Add New.
4. Click on the "Upload Plugin" button and select the zip file you downloaded.
5. Activate the plugin.

Configuration
1. Navigate to the plugin settings under Settings > Mailchimp Sync.
2. Enter your Mailchimp API key and List ID.
3. Save the settings.

Usage
Once configured, the plugin will automatically sync user data to Mailchimp whenever a new user registers or an existing user updates their profile on your WordPress site.
